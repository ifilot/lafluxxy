# LaFluxxy

## Purpose
LaFluxxy is a simple GUI for performing reaction-diffusion integration on a rectangular grid, with and without mazes.

## Dependencies



## Compilation

```bash
mkdir build
cd build
qmake ../lafluxxy.pro
make -j9
```

### Debian
```
sudo apt install qt5-default qt5-qmake libeigen3-dev libboost-all-dev
```
